Platform features

Everything you need,
nothing you don't

MICHWAR was built from the ground up for Algeria — fast geohash matching, DZD pricing, and full offline-sync.

Geohash matching

Real-time proximity search using geohash range queries finds the nearest available driver in seconds — no full-collection scans.

Secure authentication

Firebase Auth with math CAPTCHA, rate-limiting Cloud Functions, and per-field character limits stop abuse before it starts.

DZD wallet system

Drivers maintain a prepaid wallet in Algerian dinars. Commission is auto-deducted server-side with full transaction history.

Loyalty rewards

Passengers earn MICHWAR Points on every completed ride, unlocking Standard, Eco, and Premium loyalty tiers over time.

Dual-role accounts

One account, two modes. Users can act as a passenger and activate driver mode after completing document verification.

In-app SOS

One tap sends the rider's live GPS coordinates and ride context straight to the admin team — no phone call needed.
